# CEG AI Marketplace

Internal AI skills marketplace for ServiceNow CEG workflows.

This scaffold is a React + TypeScript + Vite app with a first-pass marketplace experience:

- searchable skill catalog
- category filters
- selected skill detail panel
- governance and trust metadata
- responsive enterprise app shell

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
